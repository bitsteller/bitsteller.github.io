# osm-isometric-3d

Webfrontend and rendering script for generating isometric 3D maps of OSM data using osm2pov and povray.

The code is licensed under the GPL license and can also be found over at Github (opens new window).

More information: bitsteller.bplaced.de/osm (opens new window)

The map was also featured in this article on WIRED: Hey, Apple: Please do your design magic on iOS 6 Maps (opens new window)
